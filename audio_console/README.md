Designed an audio console on Xilinx Artix-7 with SPI joystick interface and I2S2 audio module. 
Implemented volume-controlled LED animations, moving average filter, LFO modulation, and 
stereo panning. Focused on algorithm and pipeline optimization for high-frequency clock operation. 
Designed system using Vivado Block Design with AXI4-Stream interconnections between modules.

uploaded only .srcs and .xpr. it needs the joystick to be plugged on the board.
