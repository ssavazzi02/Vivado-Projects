# FPGA Projects – Xilinx Artix-7

Collection of FPGA-based digital systems developed as academic projects using Xilinx Artix-7 and Vivado.

---

 1. Audio Console – Artix-7

Real-time audio processing system with different possible effects implemented on FPGA.

 written blocks:
- SPI joystick interface
- volume controller
- effect-controlled LED animations on joystick
- effect selector
- volume-controlled LED level animation on board
- Moving average filter
- LFO modulation
- mute controller
- sound balance controller

files of the Vivado project available in the project folder.

---

2. FPGA Image Converter – Artix-7

Image convertion from RGB to gray scale implemented on FPGA.

written blocks:
- depacketizer
- packetizer
- led_blinker
- image convolution
- rgb to gray
- bram writer

files of the Vivado project available in the project folder.
