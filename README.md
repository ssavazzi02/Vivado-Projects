# FPGA Projects – Xilinx Artix-7

Collection of FPGA-based digital systems developed as academic projects using Xilinx Artix-7 and Vivado.

---

 1. Audio Console – Artix-7

Real-time audio processing system implemented on FPGA.

 Features
- SPI joystick interface
- I2S2 audio module
- Volume-controlled LED animations
- Moving average filter
- LFO modulation
- Stereo panning

Full Vivado project available in the project folder.

---

2. FPGA Image Converter – Artix-7

Image convertion from RGB to gray scale implemented on FPGA.

written blocks:
- depacketizer
- packetizer
- led_blinker
- image convolution
- rgb to gray
- bram writer

Full Vivado project available in the project folder.
